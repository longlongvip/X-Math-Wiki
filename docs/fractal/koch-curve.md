# 科赫雪花

## 科赫曲线

科赫曲线（英语：Koch curve）是一种分形。其形态似雪花，其最早出现在瑞典数学家海里格·冯·科赫（Niels Fabian Helge von Koch）的论文《关于一条连续而无切线，可由初等几何构作的曲线》

## 科赫曲线的生成

给定线段AB，科赫曲线可以由以下步骤生成：

- 将线段分成三等份（AC,CD,DB）
- 以CD为底，向外（内外随意）画一个等边三角形DMC
- 将线段CD移去
- 分别对AC,CM,MD,DB重复1~3。

## L系统

- 字符 : F
- 常数 : +, −
- 公理 : F++F++F
- 规则:
- F → F−F++F−F
- F ：向前
- - ：左转60°
- + ：右转60°
