# 婆罗摩笈多定理

婆罗摩笈多定理指出：若圆内接四边形的对角线相互垂直，则垂直于一边且过对角线交点的直线将平分对边

## 证明

因为 $\angle AFM = \angle EMC = \angle MBC = \angle MAD$
所以 $AF = MF$
所以 $\angle FMD = 90^{\circ} - \angle AFM = 90^{\circ} - \angle \angle MAD = FDM$
所以 $FM = FD$
所以 $FA = FD$，即 $EF$ 平分 $AD$

## 婆罗摩笈多四边形是圆内接四边形

- 婆罗摩笈多四边形是圆内接四边形，且对角线垂直
