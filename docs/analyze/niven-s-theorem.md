# 尼云定理

尼云定理（Niven's theorem）说的是，在 0~90° 范围内，如果正弦函数 sin 的自变量和因变量都要求是有理数，那么答案只有

- $\sin 0^{\circ} = 0$
- $\sin 30^{\circ} = \frac{1}{2}$
- $\sin 90^{\circ} = 1$

若用弧度表示，需在 $0 \le x \le \frac{\pi}{2}$的范围内，且要求 $\frac{x}{\pi}$ 及 $\sin x$ 都是有理数。其结果是

- $\sin 0 = 0$
- $\sin \frac{\pi}{6} = \frac{1}{2}$
- $\sin \frac{\pi}{2} = 1$
